LORE / TONE OVERVIEW
--------------------
A dark-comedy, South Park / Adult Swim–style point-and-click slideshow adventure where:

- The Hardigan Brothers (HANK and JONAH) are wannabe teen detectives in Northern Virginia.
- Their undocumented neighbors, the Riveras, are suddenly raided by ICE.
- The brothers’ choices pull them into:
  - A Mexican drug cartel faction
  - US Feds (ICE, CIA, shadow “Special Projects” unit)
  - Venezuelan political operators
  - Colombian smugglers / Tren de Agua

Satire targets:
- US immigration policy / ICE optics
- War on Drugs
- Media hysteria & TikTok geopolitics
- Conspiracy culture

We’re mocking systems, institutions, and narratives — not entire nationalities or everyday people.


TARGET PLAYTIME
---------------
~30 minutes for a single playthrough, with branching routes & multiple endings.
Average route: ~10–14 scenes.


GAMEPLAY LOOP (index.js)
------------------------
1. Player chooses:
   - NEW GAME
   - CONTINUE (load from saved JSON)

2. Each “scene” is a slideshow-style card with:
   - Background art
   - Characters (slide-in/out)
   - Clickable hotspots:
     - Objects (keys, phones, laptops, doors)
     - Environmental toggles (lights, windows, TV, etc.)
     - NPCs → dialogue bubbles

3. Player chooses dialogue options / actions.

4. Choices:
   - Update FLAGS (HELPED_ICE, HELPED_NEIGHBORS, CARTEL_ALLY, etc.)
   - Unlock / block later branches.

5. Inventory items unlock new interactions.

6. Endings depend on key flag combinations.

STATE MANAGEMENT (index.js)
---------------------------
Global state object:
- currentSceneId
- inventory: [ "house_key", "neighbors_usb", ... ]
- flags: {
    HELPED_ICE,
    HELPED_NEIGHBORS,
    TOOK_CARTEL_DEAL,
    WORKING_WITH_CIA,
    CARTEL_TARGET,
    INDEPENDENT_OPERATORS,
    ...
  }

Save system:
- LocalStorage snapshot:
  {
    currentSceneId,
    inventory,
    flags,
    timestamp
  }
- CONTINUE loads last snapshot.

GAMEPLAY OBJECTIVES PER BEAT
----------------------------
INTRO (S1)
- Learn click mechanics (TV, lamp, window).
- Acquire conspiracy_notebook + house_key.
- See ICE raid across the street.

FIRST BIG CHOICE (S2)
- Call ICE / obey Mom → HELPED_ICE.
- Sneak out backdoor → HELPED_NEIGHBORS + neighbors_usb.

MIDGAME
- Use inventory to gain leverage.
- Choose alliances (cartel / CIA / independent).

CLIMAX (S9)
- Multi-faction standoff at warehouse.
- Decisions around USB → endings.
